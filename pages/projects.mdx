### DevSettings
A simple SwiftUI view to manage development settings in your SwiftUI application. [(GitHub)](https://github.com/Thieurom/DevSettings)

### JourneyLog
A simple wrapper of `os.Logger` and `OSLogStore` to log and export log entries. [(GitHub)](https://github.com/Thieurom/JourneyLog)

### Landmarks
A rebuild of Apple's "Landmarks" application using the Composable Architecture. [(GitHub)](https://github.com/Thieurom/Landmarks)

### Pilot
A very simple HTTP network layer written in Swift. [(GitHub)](https://github.com/Thieurom/Pilot)

### FootballTables
A simple iOS app shows the latest matches and standings of Europe's top 5 football leagues. [(GitHub)](https://github.com/Thieurom/FootballTables)

### FootballDataClient
Swift client for `football-data.org` API. [(GitHub)](https://github.com/Thieurom/FootballDataClient)

### SanoApp
Yet another Tic-tac-toe game, for iOS, written in Swift. The game includes a single player mode called Fight-the-Robot, which is implemented using Minimax algorithm. [(GitHub)](https://github.com/Thieurom/SanoApp)

### LobstersApp
iOS app for reading Lobsters (https://lobste.rs/) stories and comments. Written in Swift. [(GitHub)](https://github.com/Thieurom/LobstersApp)

### BlocksApp
An iOS app that shows random programming quotes. [(GitHub)](https://github.com/Thieurom/BlocksApp)
